# personalized_webPage

This is a small project meant to evaluate my ability to interact with the DOM 

## How It Works

1. User put name and the age in their respective inputs an then submits.
2. the user's data is saved in the localStorage to access later.
3. Upon submissing , a card shows up with:

    i. A dive that greets the user with its name.
   
    ii. Another div that evaluates the age and gives content accordingly.
   
   iii. last dive shows a motivational quote to encouragethe user.

## Tech Stack
- HTML
- Tailwind css
- JavaScript
